Projeto Curso Git

Projeto do curso de 'introdução ao git e controle de versões' e do curso de 'introdução ao github e comandos para trabalhar em equipe'

Descrição do curso

Esses dois cursos integram o Bootcamp "Desenvolvedor Front-end ReactJS" da Digital Innovation One e são ministrados pelo professor Eduardo Cesar.

No intuito de ser um curso que prepara o aluno para colocar em prática os conceitos estudados nas aulas, propõe-se que o aluno desenvolva e disponibilize no Github um primeiro projeto.

Descrição do meu projeto

Eu decidi aproveitar para iniciar meu repositório fazendo o registro de alguns comandos básicos que aprendi no curso e que organizei aqui em uma tabela:

Comando Finalidade
git config --global user.name 'nome' configura o nome de usuário
git config --global user.email 'email' configura o email do usuário
mkdir 'nome do projeto' cria um diretório
cd 'nome do projeto' acessa um diretório
git init inicia o diretório
touch 'nome do arquivo' cria um novo arquivo
git add 'nome do arquivo' adiciona o arquivo
git add. adiciona todos os arquivos
git log mostra o histórico de commits
git branch mostra as branchs existentes
git branch 'nome da branch' cria uma nova branch
git branch -D 'nome da branch' exclui a branch
